# 深岩银河：幸存者 游戏助手

《Deep Rock Galactic: Survivor》的成就速查 · 武器指南 · 装备图鉴工具。

打游戏时开着手机或浏览器就能随手查——哪个成就怎么解锁、哪把武器强在哪、捡到的装备有什么用，一目了然。

## 它能帮你做什么

### 成就速查
- **全量收录 300 项成就**，按官方 19 个分类组织（职业解锁 / 职业进阶 / 属性统计 / 装备 / 武器超频 / 武器专精 / 武器真专精 / 生物群系 / 武器标签 / 生物 / 资源 / 神器 / 商店 / 环境 / 伤害 / 护卫 / 异常·先锋·致命 / 耐力 / 其他动作）。
- 每张成就卡展示：**中英成就名**、**解锁条件（中英对照）**、**稀有度徽标**（普通 / 稀有）、**达成率**（真实统计值，仅生物群系类标注档位 H5 / Mastery / TrueMastery）。
- 灵活筛选：按分类多选、按稀有度（普通 / 稀有）过滤、按达成率或名称排序。
- 解锁条件里若引用了某把武器，名称会**随武器译名自动同步**，改一处全局跟着变。

### 武器指南
- **42 把武器**，归属 4 大职业（侦察兵 / 枪手 / 工程师 / 钻机手），每职业含 3 个小职业（共 12 种，中文译名取自官方 Wiki）。
- **官网四类标签**（伤害类型 / 武器家族 / 武器类型 / 射击类型），中文译名按官方 Wiki 映射。
- **主观强度评级 S / A / B / C / -**，卡面明确标注"仅供参考"，帮新手快速判断优先级。
- 每把武器附带其**黄色超频（6/12 级）与红色超频（18 级）**搭配参考，省去来回翻 Wiki。
- 筛选：按职业、评级、标签、名称排序组合过滤。

### 装备图鉴
- **数十件局内装备 / 神器**，每件同时展示 **「官网原文名 + 官网效果原文」** 与 **「中文译名 + 攻略效果解读」**，原汁原味与本土化两不耽误。
- 可按**类型**、**来源**（局内附加 / 成就解锁 / 来源未归类）筛选。
- 部分条目带 **「待定」标记**——表示该项来自游戏提取表、尚待社区核实，方便后续校正，绝不杜撰。

### 通用能力
- **全局搜索**：顶部搜索框一次覆盖成就 / 武器 / 装备，匹配中英文名、标签、解锁条件。
- **中英双语**：界面外壳与数据字段均双语呈现；切换语言时，成就条件里引用的武器名也会同步切换。
- **深浅主题**：一键在浅色 / 深色间切换，深夜开局不刺眼。
- **PWA 离线**：手机或桌面浏览器"添加到主屏幕"后，像原生 App 一样使用，**没网也能查**（数据已随应用缓存）。

## 本地运行

```bash
# 获取源码
git clone git@github.com:Is-Ming/drg-survivor-helper.git
cd drg-survivor-helper

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

构建产物位于 `dist/` 目录，用任意静态服务器（nginx、caddy 等）直接托管即可。
